## Project Overview
The Personal Expense Analytics Dashboard is an interactive Power BI project designed to analyze personal income and expenses. It provides insights into spending habits, income sources, and financial performance through dynamic visualizations and KPI cards.
### The dashboard helps users:
- Track total income and expenses.
- Monitor monthly financial trends.
- Identify the highest spending categories.
- Analyze income sources.
- Make better financial decisions using data.

## Final Conclusion
This project demonstrates how Power BI can transform raw financial transaction data into meaningful insights through interactive dashboards. By tracking income, expenses, and spending patterns, the dashboard helps users better understand their financial behavior and supports informed budgeting and financial planning. It also showcases practical Power BI skills, including KPI creation, chart design, filtering, and dashboard development.
